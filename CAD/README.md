This folder contains all of the CAD files necessary to construct the robotic hand.

Hand Assembly.SLDASM: Solidworks assembly containing the PCB, spacer, palm, mount, thumb, and one finger

Mount.SLDPRT: A Solidworks model of a mount for the hand to sit on

PCB Spacer.SLDPRT: A Solidworks model of a spacer to go between the palm and the PCB. Assembly contains 3

Palm.DXF: A DXF file of the palm that was used to design the PCB

Palm.SLDPRT: A Solidworks model of the palm, finger, and thumb

Passive Joint.SLDPRT: A Solidworks model of an attachment for the thumb joint (in Palm.SLDPRT)

RobotHandFINAL.step: A STEP file of the final PCB

Spool.SLDPRT: A Solidworks model of the spool. Assembly contains 5
