bool InitComms ( )

Initialize ESP-NOW communications and register the receiver peer.

Initialize ESP-NOW communication subsystem.

This function:

  • Puts WiFi into station mode.
  • Initializes ESP-NOW.
  • Registers OnDataSent as the send callback.
  • Adds the receiver peer specified by ::BROADCAST_ADDRESS.
Returns
true on success, false if initialization or peer registration fails.

◆ ReadQueue()

uint16_t ReadQueue ( QueueHandle_t queue)

Empty a queue and return the average of all readings.

This function repeatedly calls ::xQueueReceive with zero block time until the queue is empty. It computes the average of all values removed.

Parameters
queueHandle to the FreeRTOS queue containing sensor readings.
Returns
Average of all readings in the queue, or 0 if the queue was empty.

◆ ReadSensorsTask()

void ReadSensorsTask ( void * p_params)

FreeRTOS task that continuously reads flex sensors and pushes values into per-finger queues.

The task:

  • Configures ADC resolution and attenuation.
  • Periodically reads each sensor using SendToQueue.
  • Logs an error over Serial if queue send fails.
Parameters
p_paramsPointer to task parameters (unused).

Read a single ADC value and send it to a FreeRTOS queue.

◆ TransmitTask()

void TransmitTask ( void * p_params)

FreeRTOS task that averages queue data for each finger and sends packets over ESP-NOW.

The task:

  • Reads and averages queued sensor values with ReadQueue.
  • Packages readings into a packet.
  • Prints readings via PrintReadings.
  • Sends the packet with ::esp_now_send and reports success or failure.
Parameters
p_paramsPointer to task parameters (unused).
